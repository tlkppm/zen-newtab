# Chrome/Edge 浏览器美化插件产品需求文档 (PRD)

## 1. 项目概述
本项目旨在开发一款适用于 Chrome 和 Edge 浏览器的极简主义新标签页美化插件。
**核心设计原则**：极简、纯净、无干扰。
**明确禁止**：不使用渐变色，不使用赛博朋克风格。

## 2. 核心功能
### 2.1 全屏背景
- **功能描述**：新标签页背景展示高质量的静态图片。
- **图片来源**：内置精选风景/自然图片，或接入 Unsplash API（可选）。
- **样式**：全屏铺满，可叠加轻微的遮罩层以保证文字可读性，但遮罩层应为纯色（如半透明黑/白），**禁止使用彩色渐变**。

### 2.2 实时时钟
- **功能描述**：在页面中央或显著位置显示当前时间。
- **样式**：
  - 极简字体（如 Sans-serif）。
  - 纯白色或纯黑色文字，视背景亮度而定。
  - 无阴影或极轻微阴影，无发光特效。

### 2.3 极简搜索框
- **功能描述**：提供 Google/Bing 搜索功能。
- **样式**：
  - 简单的圆角矩形或下划线样式。
  - 背景为纯色（白色/半透明白色），无边框或细边框。
  - 聚焦时无复杂的动画或颜色变化，仅改变边框颜色或透明度。

### 2.4 常用网站快捷方式 (Speed Dial)
- **功能描述**：显示用户常访问的网站图标。
- **样式**：
  - 简单的网格布局。
  - 图标使用网站 Favicon 或简单的首字母图标。
  - 悬停效果：简单的放大或背景色加深，**禁止使用霓虹灯或渐变光效**。

## 3. UI/UX 设计规范
- **色彩体系**：
  - 主色调：中性色（黑、白、灰）。
  - 强调色：单色（如纯蓝、纯绿），用于按钮或选中状态，避免高饱和度荧光色。
- **排版**：
  - 字体清晰易读，字重适中。
  - 元素之间留白充足（Padding/Margin 为 4 的倍数）。
- **交互**：
  - 响应迅速，动画使用简单的 Fade-in 或 Slide-up，避免复杂的变形或色彩流动。

## 4. 用户流程
1. 用户打开新标签页。
2. 看到全屏风景背景和中央的时钟。
3. 点击搜索框输入内容进行搜索。
4. 点击下方的快捷方式图标跳转网页。
